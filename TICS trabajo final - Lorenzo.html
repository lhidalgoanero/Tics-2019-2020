Modelo OSI
Durante este trimestre nos dedicamos a estudiar dos aspectos fundamentales, en la primera parte del trimestre estudiamos, divididos por grupos, los distintos niveles del modelo OSI.
Definición

El modelo OSI fue un intento para la creación de un estandar para que siguiran los diseñadores de nuevas redes.Explica lo que debe hacer cada componente de la red. Para una mejor organización el modelo se divide en capas y cada una de estas tiene una función determinada, y a su vez, esta ligado tanto a la capa anterior como a la posterior a ella.

Capas Nivel
Nivel Físico 1
Nivel de enlace de datos 2
Nivel de red 3
Nivel de transporte 4
Nivel de sesión 5
Nivel de representación 6
Nivel de aplicación 7
Función de cada capa
  • Nivel físico = Señal y transmisión binaria
  • Nivel de enlace de datos = Direccionamiento físico
  • Nivel de red = Determina ruta IP
  • Nivel de transporte = Conexión extremo a extremo y fiablidad de datos
  • Nivel de sesión = Comunicación entre dispositivos de la red
  • Nivel de representación = Representa datos
  • Nivel de aplicación = Servicios de red a aplicaciones
